# aituition.org pdfbox-api

[PDFBox](https://pdfbox.apache.org/) is an open-source Java library for working with PDF documents. It provides a wide range of features to create, manipulate, and extract content from PDF files.

This repository contains the API module of PDFBox, which allows developers to integrate PDFBox into their Java projects and leverage its functionalities.

## Features

- Create new PDF documents
- Manipulate existing PDF documents
- Extract text, images, and metadata from PDFs
- Add annotations, bookmarks, and watermarks to PDFs
- Split and merge PDF documents
- Encrypt and decrypt PDF files
- And much more!

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or later
- Maven

### Installation

1. Clone the repository:

   ```shell
   git clone https://github.com/wwall-hazmat/pdfbox-api.git
   
2. Build the project using Maven:
   
    ```shell
    cd pdfbox-api
    mvn install 
    
3. Include AiTution PDFBox API as a dependency in your project. You can add the following Maven dependency to your project's pom.xml file:
    ```shell
    <dependency>
      <groupId>org.aituition</groupId>
      <artifactId>pdfbox-api</artifactId>
      <version>1.0</version>
    </dependency>
